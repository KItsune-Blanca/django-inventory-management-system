Simple Inventory System

A simple Inventory Management System built using Django that allows users to manage products through CRUD operations.

Features

- Add new products
- View all products
- Update product details
- Delete products
- Product quantity tracking
- Responsive user interface using Bootstrap
- Django admin panel support

Technologies Used

- Python
- Django
- Bootstrap
- HTML
- SQLite3

CRUD Operations

Create

Users can add new inventory items to the system.

Read

Users can view all available products and their details.

Update

Users can edit existing product information such as:
- Product name
- Quantity
- Price
- Description

Delete

Users can remove products from the inventory system.
